This experiment wraps the Element84 STAC API and AWS public Sentinel-2 COGs into a browser-only tool.
Use the drawing controls to define your AOI; the app pulls the latest least-cloudy scenes, builds an
NDVI composite, and lets you export AOIs or NDVI time series for deeper analysis.
